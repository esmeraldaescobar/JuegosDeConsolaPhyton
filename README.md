JuegosDeConsolaPython

Proyecto final de lógica de programación desarrollado en Python.

Descripción

Este proyecto consiste en una colección de juegos ejecutados en consola, creados con Python como parte del proyecto final de la materia de Desarrollo de Lógica de Programación.

El objetivo fue aplicar estructuras de control, condicionales, ciclos, funciones, manejo de archivos CSV y reproducción de sonidos dentro de una aplicación interactiva.

Juegos incluidos
Adivinanza de números
Ahorcado
Piedra, Papel o Tijera
Trivia de preguntas
Tecnologías utilizadas
Python
Archivos CSV
Consola de comandos
Efectos de sonido (.mp3 y .wav)
Funcionalidades
Menú interactivo
Sistema de puntuación
Guardado de datos en archivos CSV
Reproducción de sonidos
Validación de respuestas del usuario
